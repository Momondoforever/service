Person API

Mit dieser API kann man Objekte vom Typ Person schreiben und lesen.

Base URL

http://localhost:8080/api/v2/person

Unterstützte Methoden

Get, Post, Put, Delete

Benutzung

Bei Put und Delete muss die id im Pfad angegeben werden. Beispiel: DELETE http://localhost:8080/api/v2/person/1

Bei Post und Put muss das Objekt Person als JSON übergeben werden. Beispiel: {"name": "Alex", "geb": "1999-01-01"}
